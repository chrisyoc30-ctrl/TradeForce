# TradeScore Cursor Delegation Prompts

## Quick Start

### 1. Extract this folder to your computer
- Download the ZIP file
- Extract to a folder (e.g., `C:\Users\YourName\Downloads\cursor-prompts`)

### 2. Open in Cursor using Command Line

**Windows (PowerShell):**
```powershell
cursor "C:\path\to\cursor-prompts\CURSOR_PROMPT_CRITICAL_PART1.md"
```

**Mac/Linux:**
```bash
cursor /path/to/cursor-prompts/CURSOR_PROMPT_CRITICAL_PART1.md
```

### 3. Follow the Execution Plan

**Phase 1: CRITICAL (2-3 hours)**
- Start with: `CURSOR_PROMPT_CRITICAL_PART1.md`
- Then do: `CURSOR_PROMPT_CRITICAL_PART2.md`
- Creates 4 missing pages + routing

**Phase 2: HIGH PRIORITY (4-5 hours)**
- File: `CURSOR_PROMPT_HIGH_PRIORITY.md`
- Implements Chat Router + Payments Router

**Phase 3: MEDIUM PRIORITY (3-4 hours)**
- File: `CURSOR_PROMPT_MEDIUM_PRIORITY.md`
- Adds validation + error handling + emails

### 4. Total Time: 9-12 hours
